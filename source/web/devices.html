Помощь

Статусы устройств
  • Stealer — выданы SMS, уведомления. Только перехват
  • RAT — выдан Accessibility. Полный контроль + VNC
  • Онлайн (зелёный) — устройство на связи
  • Офлайн (серый) — нет связи или выключено
VNC / RAT управление
  1. Откройте устройство со статусом RAT
  2. Нажмите кнопку RAT или блок "RAT Управление"
  3. Для работы нужно разрешение Accessibility на устройстве
Возможности: удалённый экран, тач-управление, файлы, камера, микрофон, кейлоггер, скрин-лок.
Привязка к устройству

Если включена Web-античернуха, для работы с логом нужно привязаться:

  1. Откройте устройство — увидите блюр-экран
  2. Нажмите Привязаться
  3. Все ваши действия будут логироваться
  4. При переходе на другое устройство — автоматическая отвязка
Push-фильтры
  • Откройте вкладку Push-фильтры в карточке устройства
  • Вкл все — пересылать все уведомления
  • Банки — только банковские приложения
  • Выкл все — отключить пересылку
Совет: оставьте только банки и платёжные приложения для чистого потока.
Метки и заметки
  • Цветные точки — быстрая пометка устройства (МФО, Банк, работа и т.д.)
  • Заметка — текстовое поле для личных записей по логу
  • Метки видны в таблице устройств для быстрой ориентации
