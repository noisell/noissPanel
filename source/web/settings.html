Помощь & Настройка

Настройка SMS-бота
  1. Создайте бота через @BotFather → /newbot
  2. Скопируйте токен и вставьте в Notify Bot Token
  3. Добавьте бота администратором в нужные каналы (SMS, Push, Connect, Inject)
  4. Укажите Channel ID каналов в настройках
Важно: бот работает через каналы, не в личных сообщениях. Все уведомления приходят в указанные каналы.
Настройка Push-уведомлений
  1. Push-уведомления приходят через того же SMS-бота
  2. На странице Devices → Push-фильтры можно выбрать какие приложения пересылать
  3. Кнопка "Банки" включает только банковские приложения
Совет: отключите ненужные приложения (мессенджеры, игры) чтобы не засорять бота.
Настройка Builder Bot
  1. Создайте бота через @BotFather → /newbot
  2. Вставьте токен в поле Bot Token
  3. Нажмите Сохранить — бот запустится автоматически
Канал подписки (опционально):
  1. Создайте канал в Telegram
  2. Перешлите любое сообщение из канала в @userinfobot — получите ID
  3. Вставьте ID в Required Channel ID (формат: -100...)
  4. Создайте ещё одного бота для проверки подписок
  5. Добавьте этого бота администратором в канал
  6. Вставьте его токен в Subscription Check Bot Token
Античернуха — как настроить
Telegram-античернуха:
  1. Создайте отдельного бота для перехвата SMS
  2. Вставьте токен в настройках античернухи
  3. Добавьте Telegram ID доверенных вбиверов в вайтлист
  4. Только они смогут нажать "Показать код"
Web-античернуха:
  1. Включите переключатель "Web-античернуха"
  2. Теперь при открытии устройства вбивер должен привязаться
  3. Все действия логируются (время на логе, команды)
  4. Для перехода на другой лог — нужно отвязаться
Вбиверы — добавление
  1. Откройте вкладку Вбиверы
  2. Нажмите Добавить вбивера
  3. Укажите логин и пароль
  4. Передайте логин/пароль вбиверу для входа на панель
Примечание: все вбиверы создаются с одинаковой ролью — они видят только устройства и работают с логами.
Ретрансляция SMS
  1. Создайте бота через @BotFather
  2. Вставьте токен в Retrans Bot Token
  3. Укажите номер телефона — при первом подключении устройства SMS с этого номера будут пересылаться для идентификации
  4. Укажите Channel ID канала (бот должен быть админом)
Применение: при первом подключении устройства позволяет узнать номер телефона по входящему SMS.
VNC — удалённое управление
  1. На устройстве должно быть выдано разрешение Accessibility
  2. Статус устройства изменится на RAT
  3. Кнопка RAT в карточке устройства откроет VNC
Советы:
  • Используйте авто-качество для адаптации под скорость
  • Touch-блок — блокирует нажатия жертвы
  • Экран-лок — ставит графический/PIN замок
